
What is React?
	React is a front end JS library that is designed for fast performance, better templating and rapid prototyping.
What is an Element?
	Elements are the HTML DOM elements on a webpage.
What is a Component?
	Components are custom HTML elements defined by functions or classes.
What is a Prop?
	Props are properties. These are passed into the Component as an argument.
